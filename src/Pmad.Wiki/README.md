# Pmad.Git.Wiki

`Pmad.Git.Wiki` is an ASP.NET Core library that lets host a git/markdown based wiki into an MVC application.

Features:
- Git based wiki storage
- Markdown rendering (powered by Markdig)
- Bootstrap 5 based UI
- Easy integration into existing ASP.NET Core MVC applications
- Page localization support
- Optional fine-grained access control
